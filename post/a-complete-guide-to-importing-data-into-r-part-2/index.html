A Complete Guide to Importing Data into R - Part 2

14 Jul 2017
data-wrangling
import

Introduction

This is the fourth post in the series Data Wrangling with R. In the previous post, we learnt to import data from flat files, excel spreadsheets and statistical softwares. In this post, we will learn to import data from:

  • JSON
  • XML
  • googlesheets

JSON

A lot of API data is available in the JSON format. You can use the jsonlite package to import JSON data into R.

# jsonlite::fromJSON()

XML

Some APIs offer data in the XML format. Use the xml2 package to import XML data into R.

# x <- read_xml('sample_xml')

# xml_name(x)
# xml_text(x)
# xml_attrs(x)
# xml_children(x)
# baz <- xml2::xml_find_all(x, './/book')
# xml_path(baz)
# xml_attr(baz, "id")
# xml_structure(x)
# as_list(x)

# xmlToDataFrame(x, c("character", "character", "character", "numeric",
