A complete guide to importing data into R

08 Jan 2017
data-wrangling
import / data

Introduction

This is the third post in the series Data Wrangling With R. In the previous post, we learnt about tibbles. In this post, we will learn to import data into R from different sources.

Importing data is the first step in any data analysis workflow. If you cannot import data from differernt sources or file formats, you cannot proceed with data analysis. R allows you to import data from several sources and a variety of file formats. In this post, we will focus on different file formats.

Flat Files

CSV

CSV (comma separated values) is a very popular file format. We can read data from CSV files using 2 different functions.

# read.csv()
# readr::read_csv()

If the separator is ;, use read_csv2().

# readr::read_csv2()

TSV

Tab separated values.

# readr::read_tsv()

General

Use read_delim() and specify the separator using delim argument.

# readr::read_delim(file_name, delim = '')

Table

If the columns are separated by space, use read_table() or read_table2().

# readr::read_table()
# readr::read_table2()

We can also specify column types while reading data from the files.

# col_integer(), col_logical, col_character, col_double, col_datetime, col_date, col_time

Excel

You can import .xls or .xlsx files using the readxl package.

  • specify the file name
  • specify the sheet
# readxl::read_excel()

Statistical Softwares

You can import files from other statistical softwares such as STATA, SPSS and SAS using the haven package.

STATA Files

# haven::read_stata()

SPSS Files

# haven::read_spss()

SAS Files

# haven::read_sas()
