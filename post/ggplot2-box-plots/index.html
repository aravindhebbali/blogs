ggplot2: Box Plots

21 Apr 2016
ggplot2
ggplot2 / visualization

Introduction

This is the 9th post in the series Elegant Data Visualization with ggplot2. In the previous post, we learnt how to build line charts. In this post, we will focus on boxplots.

We will continue to use the ecommerce data. You can download it from here or import directly using the readr package.

Simple Box Plot

ggplot(ecom) +
  geom_boxplot(aes(x = factor(1), y = n_visit))

Now we need to mention something for X axis and hence we use the number 1. If we do not mention anything, ggplot2 will return an error. You can generally use any categorical variable for the X axis. Below is an example:

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = n_visit))

If you want horizontal boxplots, use coord_flip().

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = n_visit)) +
  coord_flip()

Notch

Notches are used to compare groups; if the notches of two boxes do not overlap, this suggests that the medians are significantly different.

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = n_visit),
               notch = TRUE)

Jitter

Overlay points on boxplots.

ggplot(ecom, aes(x = factor(device), y = n_visit)) +
  geom_boxplot() +
  geom_jitter(width = 0.2, color = 'blue')

Outliers

Color

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               outlier.color = 'red')

Shape

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               outlier.shape = 23)

Let us extend the Y axis and see if the ouliers appear more legible.

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               outlier.shape = 23) +
  expand_limits(y = c(0, 1100))

Size

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               outlier.size = 3) +
  expand_limits(y = c(0, 1100))

Increasing the size is not very useful.

Alpha

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               outlier.color = 'blue', outlier.alpha = 0.3) +
  expand_limits(y = c(0, 1100))

Decreasing the alpha increases the visibility of the outlier points.

Box Color

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               fill = c('blue', 'red', 'green')) +
  expand_limits(y = c(0, 1100))

Map box color to variable.

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration,
               fill = factor(device))) +
  expand_limits(y = c(0, 1100))

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               fill = 'blue', alpha = 0.3) +
  expand_limits(y = c(0, 1100))

Line Color

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               color = c('blue', 'red', 'green')) +
  expand_limits(y = c(0, 1100))

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration,
               color = factor(device))) +
  expand_limits(y = c(0, 1100))

Line Width

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               size = 1.5) +
  expand_limits(y = c(0, 1100))

Line Type

ggplot(ecom) +
  geom_boxplot(aes(x = factor(device), y = duration),
               linetype = 2) +
  expand_limits(y = c(0, 1100))

Summary

In this post, we learnt to:

  • build box plots
  • modify outlier color, shape, size etc.
  • modify box color
  • modify box line color, size and line type
