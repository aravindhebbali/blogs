Working with strings in R using stringr

23 Mar 2017
data-wrangling
strings / stringr

Introduction

This is the eleventh post in the series Data Wrangling with R. In the previous post, we learnt to manipulate date and time using the lubridate package. In this post, we will learn to work with string data in R using stringr. Let us begin by installing and loading stringr and a set of other pacakges we will be using.


Libraries, Data & Code

We will use the following libraries in this post:

  • stringr
  • readr
  • dplyr
  • tibble
  • purrr
  • magrittr

All the data sets used in this post can be found here and code can be downloaded from here.

Case Study

We will do the following in this case study:

  • extract domain name from random email ids
  • extract image type from url
  • extract image dimension from url
  • extract extension from domain name
  • extract http protocol from url
  • extract domain name from url
  • extract extension from url
  • extract file type from url


Extract domain name from random email ids

mockstring$email[1:2] %>% 
  str_split(pattern = '@') %>%
  unlist() %>%
  str_subset(pattern = '\\.') %>%
  str_split(pattern = '\\.') %>%
  purrr::map_chr(extract(1))
## [1] "fastcompany" "digg"


Extract image type from image url

mockstring$imageurl[1:5] %>%
  str_split(pattern = '\\.') %>%
  purrr::map_chr(extract(3)) %>%
  str_sub(start = 1, end = 3)
## [1] "jpg" "bmp" "bmp" "png" "jpg"

mockstring$imageurl[1:5] %>%
  str_split(pattern = '\\.') %>%
  purrr::map_chr(extract(3)) %>%
  str_split(pattern = '/') %>%
  purrr::map_chr(extract(1))
## [1] "jpg" "bmp" "bmp" "png" "jpg"


Extract dimension from image url

pattern <- "[:digits:]"
mockstring$imageurl[1:5] %>%
  str_locate(pattern = "[0-9]") 
##      start end
## [1,]    23  23
## [2,]    23  23
## [3,]    23  23
## [4,]    23  23
## [5,]    23  23
  
mockstring$imageurl[1:5] %>%
  str_sub(start = 23) %>%
  str_split(pattern = '\\.') %>%
  purrr::map_chr(extract(1))
## [1] "130x183" "106x217" "146x127" "181x194" "220x123"


Extract protocol

mockstring$url[1:5] %>%
  str_split(pattern = '://') %>%
  map_chr(extract(1))
## [1] "https" "http"  "https" "http"  "https"
  
k <- mockstring$url[1:5] %>%
  str_locate(pattern = '://') %>%
  data.frame() %>%
  extract2('start') %>%
  subtract(1)

mockstring$url[1:5] %>%
  str_sub(start = 1, end = k)
## [1] "https" "http"  "https" "http"  "https"


Extract domain name

n <- mockstring$url[1:5] %>%
  str_locate_all(pattern = "/") %>%
  map_int(extract(3))

mockstring$url[1:5] %>%
  str_sub(end = n) %>%
  str_split(pattern = '\\.') %>%
  map_chr(extract(1)) %>%
  str_sub(start = k + 4)
## [1] "engadget"   "delicious"  "w3"         "indiatimes" "tumblr"


Extract extension

mockstring$url[1:5] %>%
  str_sub(end = n) %>%
  str_split(pattern = '\\.') %>%
  map_chr(extract(2)) %>%
  str_split(pattern = '/') %>%
  map_chr(extract(1))
## [1] "com" "com" "org" "com" "com"


Extract file type

# only 2 dots in the url
mockstring$url[1:3] %>%
  str_locate_all(pattern = '\\.') %>%
  map_int(nrow) %>%
  is_greater_than(2) %>%
  sum()
## [1] 0
  
# only 1 question mark in the url
mockstring$url[1:3] %>%
  str_locate_all(pattern = "[?]") %>%
  map_int(nrow) %>%
  is_greater_than(1) %>%
  sum()
## [1] 0

# location of second dot and 1 question mark
d <- mockstring$url[1:3] %>%
  str_locate_all(pattern = '\\.') %>%
  map_int(extract(2)) %>%
  add(1)

q <- mockstring$url[1:3] %>%
  str_locate_all(pattern = "[?]") %>%
  map_int(extract(1)) %>%
  subtract(1)

mockstring$url[1:3] %>% 
  str_sub(start = d, end = q)
## [1] "jsp"  "json" "json"
